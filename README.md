## Quick Start

1. **Initialize**:
   ```bash
   ./init.sh
   ```

2. **Run GUI**:
   ```bash
   ./bin/mcq-server
   ```
   Open **[http://localhost:8000](http://localhost:8000)**.

3. **Run CLI**:
   ```bash
   ./bin/mcq-cli
   ```
